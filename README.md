# ML Models

A collection of Machine Learning models implemented using Python, focusing on learning concepts from scratch and understanding how models actually work under the hood.

## What’s Inside
- Linear Regression (NumPy + Scikit-learn)
- Multiple Linear Regression
- Gradient Descent implementation
- Dataset-based experiments

## Goal
To build strong ML fundamentals by:
- Implementing models from scratch
- Comparing with library implementations
- Understanding training, cost function, and optimization

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## How to Run

git clone https://github.com/vishal-95-um/ml-models.git
cd ml-models
pip install numpy pandas matplotlib scikit-learn
jupyter notebook

ml-models/
